金融建模quiz

Posted on 2021-10-28 Edited on 2021-11-03 In 金融学习

Summary: all about building a DCF model

  • Enterprise Value EV = D + E

    • E: shares
    • D: net debt value (debt - cash)

  • 计算EV的方式：

    • 会计法：book value
      • 从左边：EV = （short-term debt + current portion of long-term debt + long-term debts + pension liabilities）-（cash+marketable securities） + other liabilities + preferred stock + minority interest + equity
      • 从右边：EV = net working capital + good will + fixed assets
    • 有效市场法：market value
      • 只需要把equity调整为market cap （=shares*price）即可
      • market value和book value的差值，来自于资产的市场价格（高于账面价值）
    • 可比公司法
      • P0 = EPS × (P/E)benchmark
      • 用EV的情况：资本结构非常不同；净利润为负
    • DCF：EV = PV(future FCFs)
      • why free: need to reinvest
    • note：WACC = **(1-tax rate)**×Cost of debt × debt% + Cost of equity × equity%

  • DCF方法

    • 区分：FCFF 和 FCFE

      • 我们说的FCF一般是FCFF （the firm），与债务无关
      • FCFE是到equity holders的现金流

    • FCFF = EBIT ×(1-tax rate) + Depreciation - Δ（current assets - current liabilities）- CapEx

    • FCFF = NI + Interest Payments × (1-tax rate) + Depre. - Δ（current assets - current liabilities）- CapEx

      • note：CapEx是gross 固定资产的变化，没有扣除折旧的那项
      • note: 这里面没有notes payable，因为不是operating的
      • note: (EBIT - Interest payments)*(1-tax)=Net Income

    • FCFE = FCFF + Net Borrowing - Interest payments × (1-tax rate)

    • FCFE = NI + D&A - Δ（current assets - current liabilities）- CapEx + Net borrowing

      • 关于net borrowing：也就是新发的debt减去已偿还的debt。注意，现金流和debt的关系，现金流为负，意味着现金流出，也就是偿还债务、net borrowing为负

  • 注意：算interest on cash/debt的时候，都是年初年末的均值乘interest rate
